# Simple AI Streamer

Simple AI streamer is a simple implementation of chat answering AI from Youtube chat and output the answer into audio.

## Running

1. Install dependencies
```bash
python -m pip install -r requirements.txt
```
2. Update config.toml
3. Run the code
```bash
python main.py
```
